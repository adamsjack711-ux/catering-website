# Pegs Easy Gourmet — Catering Website

A responsive marketing website for Pegs Easy Gourmet, a personal and private chef service based in Kamloops, BC. The site showcases catering services for events from 2 to 50 guests.

## Pages

- **Home (`index.html`)** — Hero, about, filterable menu cards, booking form, and contact section
- **Gallery (`gallery.html`)** — Filterable photo gallery with lightbox viewer

## Features

- Filterable menu cards (All / Savoury / Sweet)
- Booking inquiry form with validation
- Lightbox image gallery with captions
- Responsive hamburger navigation
- Smooth scroll with active section highlighting

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Playfair Display, Inter)
- No frameworks or dependencies
